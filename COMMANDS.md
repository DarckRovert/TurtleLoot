# Referencia de Comandos TurtleLoot

## Comandos Principales
| Comando | Alias | Descripción |
| :--- | :--- | :--- |
| `/tl` | `/turtleloot` | Abre la Ventana Principal. |
| `/tl config` | `/tl options` | Abre la ventana de Configuración. |
| `/tl reset` | | Restablece toda la configuración a los valores por defecto. |
| `/tl version` | | Muestra la versión actual del addon. |

## Accesos Directos
| Comando | Descripción |
| :--- | :--- |
| `/gdkp` | Abre directamente la pestaña GDKP. |
| `/tl sr` | Abre directamente la pestaña Soft Reserves. |
| `/tl atlas` | Abre el Navegador de Instancias (Atlas). |

## Comandos de Susurro (Para Raiders)
Estos comandos se envían por susurro al Maestro Despojador que usa TurtleLoot.

| Comando | Uso | Descripción |
| :--- | :--- | :--- |
| `!res` | `!res [Link del Item]` | Reservar un objeto específico. |
| `!cancel` | `!cancel` | Cancelar tu última reserva. |
| `!myres` | `!myres` | Consultar qué tienes reservado actualmente. |
| `!list` | `!list` | Pedir la lista de objetos disponibles (si se ha compartido). |
| `!p1` | `!p1` | Consultar tus puntos "+1" actuales. |
